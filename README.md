# Diverse Driving School Website

A modern, responsive website built for **Diverse Driving School** to showcase services, build trust with learners, and make booking driving lessons simple.

This project was created to support a real local driving school business.

---

## What This Website Includes

- Clear information about lesson types:
  - Normal lessons
  - Intensive courses
  - Block booking
- Coverage of local test centres (Avonmouth & Kingswood)
- Testimonials section
- Strong call-to-action buttons
- Contact / booking form
- Fully responsive design (mobile-friendly)

---

## Built With

- Vite
- React
- JavaScript
- CSS
- React Router
